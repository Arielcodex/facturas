2. Detección de Anomalías en Precios Podemos crear una base de datos de "Precios Promedio". Cómo funciona: Si cargás una factura de un proveedor habitual y el Total es un 30% más alto que el promedio de los últimos 3 meses para ese mismo tipo de comprobante, el sistema lanza una alerta. Sugerencia de la IA: "Atención: Este importe es inusualmente alto para este proveedor. ¿Hubo un aumento o un error de carga?" 3. Aprendizaje de Patrones de Gasto (IA Predictiva) Podemos implementar un pequeño motor de lógica que analice las fechas. Cómo funciona: La IA nota que cada día 10 de mes cargás facturas pesadas de un proveedor específico. Sugerencia de la IA: "Faltan 2 días para el pedido habitual de Droguería X. ¿Querés revisar el stock?" 4. Clasificación Automática por OCR Podemos potenciar el Tesseract.js que ya tenemos para que no solo lea el "Total", sino que busque palabras clave como "Vencimiento", "Lote" o "Bonificación". La mejora: La IA clasifica sola si la factura tiene notas de crédito pendientes de aplicación.
